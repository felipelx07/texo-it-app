# Texo IT App

API RESTful para possibilitar a leitura da lista de indicados e vencedores
da categoria Pior Filme do Golden Raspberry Awards.


Esta aplicação foi desenvolvida utilizando:  

- **Java 19**
- **Quarkus v.3.7.2**
- **Banco de dados em memória H2.**


### Instalação
```
.\mvnw install
```
### Execução

```
.\mvnw quarkus:dev
```

### Testes
```
.\mvnw test
```
